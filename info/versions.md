### Сервер ПОЛНОСТЬЮ НЕ поддерживает:
- Версии старше 1.20.6 Java
- Версии старше 1.21.90 Bedrock

### Поддержка может быть ограничена для:
- Bedrock игроков
- Игроков с лаунчеров, предоставляющих Java на Android

### Поддержка при столкновении с ошибками НЕ оказывается:
- Игрокам TLauncher, KLauncher и любых других лаунчеров, кроме лицензионного или [FOSS*](#foss) лаунчеров.
-. Игрокам с OptiFine (https://forum.quiltmc.org/t/why-you-shouldnt-use-optifine/78)

### Приоритетную поддержку получают:
- Игроки, имеющие лицензионную копию Майнкрафта
- Игроки не имеющие наказаний в прошлом
- Игроки с титулом ВИП
- Игроки, которые предоставляют больше всего информации о своей проблеме

<span id="foss"></span>
*FOSS — **Free and Open Source Software** — имеются ввиду лаунчеры с открытым исходным кодом и предоставляемые бесплатно
